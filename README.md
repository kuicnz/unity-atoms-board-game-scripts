# unity-atoms-board-game-scripts

This project was originally inspired by an assignment from the 2017 COMP2129 course at the University of Sydney. The assignment was to implement the game in the terminal. I thought it was fun, and since I was learning Unity at the time, I recreated it as a Unity project for practice. Unfortunately, I lost the project files over the years, and only the scripts remain.

这个项目最初来源于悉尼大学 2017 年 COMP2129 课程的一次作业。作业要求是在终端中实现这个游戏。我觉得这个游戏很有趣，当时又正好在学习 Unity，于是便把它用 Unity 重做了一遍，作为练习项目。很遗憾，这些年来项目文件已经遗失，只剩下代码脚本保留了下来。
